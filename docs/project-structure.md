# Project Structure

A standard novel project using this methodology looks like this:

```text
my-novel/
├── memory/
│   ├── constitution.md
│   └── personal-voice.md
├── tracking/
│   ├── plot-tracker.json
│   ├── character-state.json
│   ├── relationships.json
│   ├── timeline.json
│   └── validation-rules.json
├── stories/
│   └── my-story-name/
│       ├── specification.md
│       ├── creative-plan.md
│       ├── tasks.md
│       └── content/
│           ├── chapter-01.md
│           └── chapter-02.md
└── knowledge/
    ├── world-setting.md
    ├── locations.md
    ├── character-profiles.md
    └── character-voices.md
```
