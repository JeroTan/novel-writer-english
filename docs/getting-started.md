# Getting Started

Welcome to Novel Writer English! This guide will help you begin your first AI-assisted novel project using our seven-step methodology.

## 1. Project Initialization
Create a new folder for your novel. We recommend copying the `templates/` folder from this repository into your new folder to get the default structure:
- `memory/`
- `tracking/`
- `knowledge/`

## 2. Step 1: Constitution
Open your AI tool and use the `constitution` agent or command. It will ask you questions to help establish your core themes, tone, and absolute rules. The output will be saved to `memory/constitution.md`.

## 3. Step 2: Specify
Use the `specify` agent. You'll provide a logline, and it will help you expand it into a full `specification.md` document for your story.

## 4. Moving Forward
Once your spec is complete, you will use `clarify`, `planner`, and `task-manager` to build your scene-by-scene outline. Then, you're ready to `write`!
