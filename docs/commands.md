# Commands Reference

If you are not using an IDE that supports `.agent.md` files, you can manually use the markdown files in the `commands/` directory as prompts for your AI.

## Core Workflow Commands
- `constitution.md`: Step 1
- `specify.md`: Step 2
- `clarify.md`: Step 3
- `plan.md`: Step 4
- `tasks.md`: Step 5
- `write.md`: Step 6
- `analyze.md` & `checklist.md`: Step 7

## Tracking Commands
- `track-init.md`: Initializes JSON files.
- `track.md`: Updates state after a chapter.
- `timeline.md`: Specific updates to the chronology.
- `relations.md`: Updates character dynamics.

## Utility
- `expert.md`: Deep dives into specific domains.
