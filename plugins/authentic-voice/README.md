# Authentic Voice Plugin

This plugin provides advanced commands and expert personas to strip away typical AI "tells" from generated prose. It replaces the older "anti-ai" modules with a positive focus on human authenticity.

## Contents
- `commands/authentic-voice.md`: A rewriting command to humanize text.
- `commands/authenticity-audit.md`: Analyzes text for common AI cliches.
- `experts/authentic-editor.md`: An expert persona for line editing.
