# Web Serial Style Guidelines

## Objective
To write highly engaging, fast-paced prose optimized for serial publication (e.g., Royal Road, Wattpad).

## Rules
1. **The Hook:** Every chapter must start immediately with action or an intriguing question.
2. **White Space:** Keep paragraphs very short (1-3 sentences). Readers are likely on mobile devices.
3. **Constant Progression:** Every chapter must either advance the plot, level up the character (literally or figuratively), or reveal new lore. No filler chapters.
4. **The Cliffhanger:** Every chapter must end on a hook—a new threat, a sudden realization, a system prompt, or an unresolved conflict that forces the reader to click "Next Chapter."
