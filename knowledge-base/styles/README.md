# Styles

This directory contains style guides that can be referenced in your `constitution.md` to dictate the prose style of your novel.

## Available Styles

- **`conversational.md`**: Natural, conversational English prose. Short sentences. Dialogue-heavy. Show-don't-tell. Think: easy reading, contemporary fiction voice.
- **`literary.md`**: Focuses on thematic resonance, complex character studies, and elevated prose without being overly ornate.
- **`minimal.md`**: Hemingwayesque. Direct, unadorned prose. Relies heavily on subtext and the "iceberg theory."
- **`ornate.md`**: Rich, ornate English prose. Longer sentences, literary vocabulary, poetic imagery. Think: Tolkien, Rothfuss, literary fantasy.
- **`web-serial.md`**: Web serial fiction conventions. Cliffhangers, fast pacing, reader engagement hooks, chapter-end teasers. Adapted for platforms like Royal Road, Wattpad, Kindle Vella.
