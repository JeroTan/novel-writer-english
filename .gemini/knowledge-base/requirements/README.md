# Knowledge Base Requirements

This directory contains rule sets for specific narrative requirements and emotional tones.

- **`authentic-voice.md`**: Rules for avoiding AI-sounding prose.
- **`fast-paced.md`**: Rules for action and high-tension scenes.
- **`romance-angst.md`**: Rules for high-drama, painful romantic arcs.
- **`romance-sweet.md`**: Rules for cozy, low-stress romantic arcs.
- **`serious-literature.md`**: Rules for thematic, introspective prose.
- **`strong-emotion.md`**: Rules for maximizing the impact of emotional scenes.
